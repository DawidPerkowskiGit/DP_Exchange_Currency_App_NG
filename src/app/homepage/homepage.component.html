Currency exchange rates application

Currency exchange rates application which uses REST API to fetch data. Users can check the available currencies and inquire about the locations they can be used in. Including viewing the latest exchange rates, users can also change the base EUR currency to other 30+ available currencies. Users can also view historical exchange rates.
